## BF_Compile 已知bug：

请不要输入太长的程序（比如大几百行）
如果需要输入的，在第一个输入（,）前请多添加一个（换行会被当作输入）

*示例：*

,[>+++<-]>.,.变为,,[>+++<-]>.,.

## BF_Compile_Command 输入方法：

在cmd.exe或类似命令行中进入其目录，输入BF_Compile_Command.exe或BF_Compile_Command，后面加上代码路径

*示例：*

**BF_Compile_Command C:\Example.bf**

*注：当代码与解释器路径相同时，无需完整路径，只需文件名*
